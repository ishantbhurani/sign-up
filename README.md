## Run Locally

Clone the project

```bash
  git clone git@github.com:ishantbhurani/sign-up.git
```

Go to the project directory

```bash
  cd sign-up
```

Run `index.html`

```bash
  <browsername> index.html
```

E.g.

```bash
  firefox index.html
```

```bash
  google-chrome index.html
```
